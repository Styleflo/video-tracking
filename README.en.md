# Object detection and tracking in videos

***Computer Vision and Image Processing (8INF804) & Deep Learning (8INF887)***  
***Université du Québec à Chicoutimi (UQAC) - Winter 2026***

## 📖 Project description

### Our project is an AI application capable of detecting and tracking one or more objects within a video sequence. 


## 🚀 Features

* **Object detection**: Real-time detection of objects in a scene.
* **Continuous tracking**: Maintaining the identification and tracking of persistent objects throughout the video.
* **Performance evaluation**: Measuring the application’s effectiveness using appropriate metrics such as IoU, mAP, and F1-score.

## 🛠️ Technological Tools

* **Language**: Python.
* **Detection Models (Vision/AI)**: YOLO26 (considering: RetinaNet, Faster R-CNN or DETR).
* **Tracking Algorithms**: DeepSORT, ByteTrack.
* **Validation Tools**: Implementation of hyperparameter search methods and comparative evaluation on datasets.
